# SwiftWebServer Demo

This is a multi-platform Xcode Demo app for `SwiftWebServer` and the optional `SwiftWebServerWebUpload` module.

## Generate the Xcode project

The `.xcodeproj` is generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `project.yml`.

```bash
cd Demo
xcodegen generate
```

## Run the Demo

```bash
open SwiftWebServerDemo.xcodeproj
```

Then select the target for your platform (iOS, macOS, tvOS, or visionOS) and run.

## Features shown

- Start / suspend / resume / stop the server
- Basic routing: `/hello`, `/json`
- Basic authentication: `/admin`
- WebUpload: open `http://localhost:8080/upload` in a browser to upload files
- Native uploaded-file list: tap **View Uploaded Files** in the app
